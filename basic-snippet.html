Tags in the snippet

  • Paragraph tag / p tag
  • Anchor tag / a tag

Online references

Relationship between tags in terms of nesting

a is a child of p and p is a parent of a.

Relationship between p and /p

p signals where the text for the paragraph begins, and /p signals where the text ends.

Attribute names on the a tag

  • href (hyperlink reference)
  • id
  • class
  • target

Values for each attribute

  • href: https://google.com
  • id: the-best-link
  • class: banana
  • target: _blank

Purpose and effect of each attribute/value pair

  • href: https://google.com - links the text within the a tags - in this case it's to https://google.com.
  • id: the-best-link - gives the element a unique individual name - in this case it's the-best-link.
  • class: banana - allows the element to be grouped with similar elements by giving their group a name - in this case it's banana.
  • target: _blank - Specifies where the link in the a tags should be opened - in this case it's in a new window.
